## Instructions

### Setup

```shell
git clone https://github.com/codelikeagirl29/gitprofile
cd gitprofile
```

and then to install and view in browser.

```shell
npm install
npm run start
```

Install and run -

```shell
yarn
yarn start
```

### How to use

- Open `src/App.js` and make changes.

- Build and deploy -

```shell
yarn build
yarn deploy
```
